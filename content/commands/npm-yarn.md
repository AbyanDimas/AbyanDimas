## Install dependencies

**NPM**:

```bash
npm install
```

**Yarn**:

```bash
yarn install
```

## Install specific package

**NPM**:

```bash
npm install express
```

**Yarn**:

```bash
yarn add express
```

## Install as dev dependency

**NPM**:

```bash
npm install --save-dev jest
```

**Yarn**:

```bash
yarn add --dev jest
```

## Uninstall package

**NPM**:

```bash
npm uninstall lodash
```

**Yarn**:

```bash
yarn remove lodash
```

## Update all packages

**NPM**:

```bash
npm update
```

**Yarn**:

```bash
yarn upgrade
```

## Run script from package.json

**NPM**:

```bash
npm run dev
```

**Yarn**:

```bash
yarn dev
```

## Global install

**NPM**:

```bash
npm install -g typescript
```

**Yarn**:

```bash
yarn global add typescript
```

## Check outdated packages

**NPM**:

```bash
npm outdated
```

**Yarn**:

```bash
yarn outdated
```

## Clear cache

**NPM**:

```bash
npm cache clean --force
```

**Yarn**:

```bash
yarn cache clean
```
